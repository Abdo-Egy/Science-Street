Simple Audio Library is a runtime and cross-platform package for Unity for loading, decoding, encoding and converting audio clips.
It also provides additional features for modifying audio clips (cut, trim, change sampling rate), reading ID3 tags and creating histograms.

How to start:
	- open and run [Demo] scene
	- press buttons with examples
	- look at code examples near the buttons
